# Devpod Setup

This repository contains scripts to quickly configure a new Devpod environment with Zsh, Oh My Zsh (including syntax highlighting, autosuggestions, and tab-title plugins), tmux aliases, and more.

## Usage

Run the setup script:

```bash
sh setup.sh
```

## SSH Key Generation

To generate a new SSH key for use with GitHub or other services, run:

```bash
ssh-keygen -t ed25519 -C "$GOOGLE_EMAIL"
```
